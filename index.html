Project Brief

CIATS: Cloud Inventory & Asset Tracking System

Revisi Dokumen: Versi 3-Role (Admin, Operator, Karyawan)

1. Ringkasan Proyek

CIATS adalah aplikasi manajemen inventaris berbasis web yang dirancang untuk memodernisasi pencatatan aset perusahaan. Sistem ini menggantikan metode manual dengan database terpusat yang mendukung pelacakan real-time dan request peminjaman mandiri oleh karyawan.

Backend: Laravel 10/11 Database: Firebase (NoSQL) Frontend: Blade & JS UI: Bootstrap/Tailwind

3. Fitur Utama & Alur Kerja

A. Manajemen Aset & QR Code

Setiap aset yang didaftarkan akan otomatis mendapatkan QR Code unik. QR Code ini digunakan untuk:

  • Identifikasi cepat via scan kamera HP/Scanner.
  • Direct link ke halaman detail aset.

B. Alur Peminjaman (Self-Service)

  1. Karyawan: Login -> Cari Barang -> Klik "Request Peminjaman".
  2. Operator: Menerima notifikasi -> Menyiapkan Barang -> Scan QR -> Ubah status jadi "Dipinjam".
  3. Sistem: Mengurangi stok tersedia & mencatat riwayat peminjaman.

C. Pelaporan & Audit

Menyediakan ekspor data ke format PDF/Excel untuk keperluan stok opname bulanan dan laporan aset.
