2. Pembagian Tugas & Panduan Teknis

Berikut adalah detail apa yang harus dikerjakan oleh masing-masing anggota mulai hari ini:

üë§ Anggota 1 (Backend Core) - DONE ‚úÖ

  • Setup Project Laravel & Koneksi Firebase (Selesai).

  • Menyediakan route dasar /assets untuk tim lain.
  • Next Task: Membantu Anggota 3 mengintegrasikan logika transaksi.
üë§ Anggota 2 (Frontend & UI)
Blade View

Tugasmu mempercantik tampilan yang dibuat Anggota 1 dan membuat tampilan baru.

  • Install Template Admin: Cari template HTML gratis (AdminLTE/Stisla), lalu copy assets (CSS/JS) ke folder public.
  • Layouting: Buat file layouts/app.blade.php sebagai kerangka utama (Sidebar, Navbar, Footer).
  • Halaman Index: Ubah tampilan tabel di assets/index.blade.php agar terlihat profesional (pakai DataTables jika bisa).
  • Mobile Responsiveness: Pastikan tabel dan tombol bisa dibuka enak di HP (penting untuk fitur scan nanti).
üë§ Anggota 3 (Tracking & QR)
Advanced Logic

Kamu fokus di logika perpindahan barang.

  • Transaction Controller: Buat controller baru untuk menangani logika "Peminjaman".
    Logika: Saat dipinjam, update status di node 'assets' jadi 'in_use', lalu catat riwayat di node 'transactions'.
  • QR Code: Install library simplesoftwareio/simple-qrcode. Tampilkan QR Code di halaman detail barang.
  • Fitur Search: Buat fitur pencarian barang berdasarkan serial number atau nama.
üë§ Anggota 4 (Auth & Report)
Security & Output

Kamu fokus mengamankan pintu masuk dan output data.

  • Login System: Buat halaman Login. Cek email & password user ke node users di Firebase.
  • Role Middleware: Buat middleware untuk membedakan 'Admin' dan 'Operator'.
    Admin boleh hapus data, Operator tidak boleh.
  • Create User (Admin Only): Buat fitur agar Admin bisa menambah user baru (Operator/Karyawan). Ingat: Tidak ada fitur Register untuk publik.
  • Export PDF: Install barryvdh/laravel-dompdf untuk mencetak laporan stok.
