CIATS

Cloud Inventory & Asset Tracking System

Sistem Manajemen Inventaris & Pelacakan Aset Perusahaan

1. Ringkasan & Tujuan

Proyek ini bertujuan membangun aplikasi berbasis web untuk menggantikan pencatatan aset manual. Sistem ini akan mengelola siklus hidup aset perusahaan (Laptop, Furniture, Kendaraan) mulai dari pengadaan, peminjaman (tracking), hingga penghapusan aset.

Tantangan utama adalah sinkronisasi data secara Real-time agar stok selalu akurat saat diakses oleh banyak staff sekaligus.

2. Teknologi (Tech Stack)

Laravel 10/11 Backend Framework
Firebase Realtime Database / NoSQL
Blade & Bootstrap Frontend UI/UX
QR Code Asset Identification

5. Alur Kerja (Workflow)

  1. Input: Staff scan barang baru atau input manual -> Data tersimpan di Firebase.
  2. Labeling: Sistem mencetak QR Code -> Ditempel di fisik aset.
  3. Tracking: Saat barang dipinjam, Staff scan QR -> Update status jadi "In Use" -> Lokasi berubah.
  4. Monitoring: Admin melihat perubahan stok dan lokasi aset secara real-time di Dashboard.
